Multi-seed leaderboard

11 synthetic worlds (seeds 101, 202, 2025, 303, 404, 505, 606, 707, 77, 808, 909) — which engines are reliably good, not just lucky on one. This is the canonical ranking; the single-dataset leaderboard is a per-seed deep dive with the full engine set and figures. run tracker

A single leaderboard answers "who won this dataset?"; that ranking is unstable. This page answers "who wins on average, and how often?" — the only fair way to compare methods. Average rank and win-rate are computed among non-diverged national engines within each run; MAE spread (std across seeds) is the stability we actually care about; ESS / R̂ is sampling quality for the Bayesian engines (a recovery you couldn't sample isn't a recovery).
